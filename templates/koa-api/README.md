## Documentation

- run : `$ npm install`
- run : `$ npm run doc`
- open `./apidoc/index.html` to read the documentation

## Tests

- run : `$ npm install`
- run : `$ npm test`
- run : `$ npm run watch` : It launch test when you change the code

## Code Coverage

After running `$ npm test` you can access the code coverage report here : `./coverage/index.html`
